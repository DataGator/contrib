Launch development web server
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block::

    $ python manage.py runserver 127.0.0.1:8000

More information on django project management can be found at `<https://docs.djangoproject.com/en/1.8/ref/django-admin/>`_.
